[Download For Windows]()

[Download For macOS]()

[Download For Linux]()

### Build Setup

``` bash
# clone
git clone https://github.com/Chakery/translate.git
cd translate

# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron app for production
npm run build

```
